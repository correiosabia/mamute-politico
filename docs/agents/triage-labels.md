# Triage Labels

Use this label vocabulary for GitHub Issues:

| Role | Label |
| --- | --- |
| Maintainer needs to evaluate | `needs-triage` |
| Waiting on reporter or stakeholder context | `needs-info` |
| Ready for an AFK coding agent | `ready-for-agent` |
| Ready for human implementation | `ready-for-human` |
| Will not be actioned | `wontfix` |

Issues produced from approved PRDs or TechSpecs should default to `ready-for-agent` unless they require a product or architectural decision first.
