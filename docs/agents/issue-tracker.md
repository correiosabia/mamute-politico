# Issue Tracker

This repository tracks implementation work in GitHub Issues:

- Repository: `correiosabia/mamute-politico`
- Preferred CLI target: `--repo correiosabia/mamute-politico`

When creating issues, use vertical slices that are independently implementable and verifiable. For crawler work, include:

- Source URL or endpoint family.
- Target table or API behavior.
- Expected idempotency rule.
- Fixture and test expectations.
- Production validation query or freshness check.

Do not close issues based on title similarity. Close only when implementation, tests, and validation evidence match the acceptance criteria.
